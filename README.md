# userdata-api

API for fetching user data

# Prerequisite

- node v8
- yarn

# Installation

## All environment

Install node modules (dependencies)

```
$ yarn
```

# Run application

## Development

Start app

```
$ yarn start
```

Debugging

```
$ yarn run dev
```

## Staging / Production

Build project

```
$ yarn run build
```

Run application (Staging/Production)

```
$ export NODE_ENV={staging|production} /path/to/node /path/to/project/index.js
```
